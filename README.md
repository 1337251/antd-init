# antd-init

[![NPM version](https://img.shields.io/npm/v/antd-init.svg?style=flat)](https://npmjs.org/package/antd-init)

[Ant Design](https://github.com/ant-design/ant-design) boilerplate generator.

----

## Install

```bash
$ npm i antd-init --save
```

## Usage

```bash
$ mkdir foo && cd foo
$ antd-init
```
